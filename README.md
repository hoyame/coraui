# FiveM-CoraUI
Library written in typescript to create menus on FiveM

# Lua : https://github.com/hoyame/FiveM-CoraUI/tree/lua

# Discord : https://discord.gg/Em3Fn9Amyv

# Features

- Color Panel
- Slidebar panel
- Slider panel
- Straight text
- Glare in the header
- Submenus
- Heritidy Panel

# Example
